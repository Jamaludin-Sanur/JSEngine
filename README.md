Requirement :
NodeJS

Installation :
- go to 'cli' folder
- run npm i
- go to 'core' folder
- run npm i
- run npm start

Execution :
- go to 'cli' folder (contain 'cli.js')
- allow file 'cli.js' to be executed.
  In unix system you can use 'chmod +x cli.js'
- open terminal to 'cli' directory (contain 'cli.js') then run './cli.js'
